# Installation

To install this package you need to do the following steps:

1. In your projects composer.json file add the following

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/untitled-developers/kockatoos-admin-core"
        }
    ]
}

```

```json
{
    "require": {
        "untitled-developers/kockatoos-admin-core": "^1.0"
    }
}
```
